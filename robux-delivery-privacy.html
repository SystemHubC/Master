RobuxDelivery Privacy Policy

RobuxDelivery processes FunPay order metadata and buyer messages required to automate Robux delivery.

Data sent to Swizzyer/RBCode

The plugin may send the Roblox username provided by the buyer, selected Robux package/items, FunPay order id/chat id as metadata, and delivery status polling requests to the configured Swizzyer/RBCode API endpoint.

Hosted Link mode

By default, the plugin does not ask the buyer to send a Roblox password in FunPay chat. It uses Hosted Link mode: the buyer completes verification/2FA on the provider verification page.

Stored data

The plugin stores operational state in FunPay Pulse storage: order ids, chat ids, selected Roblox username, external order id, delivery status and timestamps.

Secrets

API keys and Telegram bot tokens are stored in FunPay Pulse secret storage and are not included in normal configuration exports.
